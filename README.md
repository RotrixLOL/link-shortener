# link shortener

![Preview](https://github.com/RotrixLOL/link-shortener/blob/main/screenshot.png)

My link shortener made with React, Tailwind, TypeScript and ShrinkMe API

## Installation

- Run `npm i`
- Run `npm run client:dev`
- Success!
